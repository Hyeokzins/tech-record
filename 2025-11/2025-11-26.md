## 배운 점 및 인사이트

### Low 트리거 릴레이의 작동 원리 이해

**직관적인 오해:**
- LOW라는 용어 때문에 릴레이를 끄는 것처럼 보였음
- HIGH가 켜는 것, LOW가 끄는 것이라고 생각했음

**실제 동작 원리:**
- 릴레이는 **전압 차이**로 작동함
- 릴레이 VCC: 5V (고정)
- 릴레이 In: 0V 또는 5V (Arduino에서 제어)

**LOW(0V) 입력 시:**
```
VCC(5V) - In(0V) = 5V 전압 차이 발생
    ↓
전류가 흐름
    ↓
릴레이 코일 작동
    ↓
릴레이 ON
```

**HIGH(5V) 입력 시:**
```
VCC(5V) - In(5V) = 0V 전압 차이 없음
    ↓
전류가 흐르지 않음
    ↓
릴레이 코일 정지
    ↓
릴레이 OFF
```

**핵심 깨달음:**
- 용어(LOW/HIGH)와 실제 동작(ON/OFF)이 반대로 보일 수 있음
- 전압 차이 원리를 이해하니 **LOW = ON, HIGH = OFF**가 명확해짐
- 이런 릴레이를 "Low 트리거 릴레이"라고 부름

### 기타 학습 내용
- **디지털 신호의 실제 전압**: LOW = 0V, HIGH = 5V
- **멀티미터 측정**: DC 어댑터는 V- (DC 모드)로만 측정
- Python 시리얼 통신으로 Arduino에 명령 전송 가능

## 문제 해결 및 코드 예시

### Arduino 기본 릴레이 제어 코드
```cpp
void setup() {
  Serial.begin(9600);
  pinMode(8, OUTPUT);
  digitalWrite(8, HIGH);  // 초기: HIGH → 릴레이 OFF
}

void loop() {
  if (Serial.available()) {
    char command = Serial.read();
    
    if (command == '1') {
      digitalWrite(8, LOW);   // LOW(0V) → 릴레이 ON → 9V 출력
    } 
    else if (command == '0') {
      digitalWrite(8, HIGH);  // HIGH(5V) → 릴레이 OFF → 0V 차단
    }
  }
}
```

### Python 시리얼 통신 코드
```python
import serial
import time

# 시리얼 통신 초기화
arduino = serial.Serial('COM6', 9600, timeout=1)
time.sleep(2)

# 릴레이 제어
arduino.write(b'1')  # '1' 전송 → Arduino LOW → 릴레이 ON
time.sleep(2)
arduino.write(b'0')  # '0' 전송 → Arduino HIGH → 릴레이 OFF

arduino.close()
```

### 릴레이 동작 정리

| Arduino 명령 | 핀 출력 | 전압 | 전압 차이 | 릴레이 상태 | 출력 전압 |
|-------------|--------|------|----------|-----------|----------|
| digitalWrite(8, LOW) | LOW | 0V | 5V - 0V = 5V | **ON** | 9V |
| digitalWrite(8, HIGH) | HIGH | 5V | 5V - 5V = 0V | **OFF** | 0V |

### 회로 연결
```
Arduino 5V → 릴레이 VCC
Arduino GND → 릴레이 GND
Arduino 8번 핀 → 릴레이 In
9V 어댑터 (+) → 릴레이 COM
릴레이 NO → LED (+)
LED (-) → 9V 어댑터 (-)
```

### 핵심 요약
- Python에서 0 또는 1을 전송하여 Arduino를 통해 릴레이 제어 가능
- Low 트리거 릴레이는 전압 차이 원리로 작동 (LOW = ON, HIGH = OFF)
- 이 원리를 이해하면 직관과 반대되는 동작을 명확히 이해할 수 있음
