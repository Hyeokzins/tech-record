현재 com 포트 조회하는 기능이 있고 그것을 통해 찾을수있음 
- 이때 usb에 연결된 디바이스를 찾는게아닌 com으로 연결된것을 찾는것
- com 연결(쌍방향 통신가능:아두이노),hid(유저 입력만 받는것:키보드,마우스등)
- 그래서 아두이노만 타겟할수있음 사실상 이런 코드를 기존 아두이노컨트롤러 코드에 삽입해서 내부코드에서 알아서 찾게 만들어 연결하게함
- 애초에 포트번호가 안바뀌게도 조치를했고 포트번호를 몰라도 연결할수있게 조치함 2단으로

##  문제 해결 및 코드 예시
### 코드 예시 1:
```python
def _find_arduino_port(self):
        """아두이노 포트 자동 검색"""
        ports = serial.tools.list_ports.comports()
        
        for port in ports:
            hwid = port.hwid.lower()
            desc = port.description.lower()
            
            if 'usb' in hwid and 'bluetooth' not in desc:
                return port.device
        
        return None
        
```
